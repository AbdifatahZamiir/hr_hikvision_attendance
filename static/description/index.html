Hikvision Attendance Integration for Odoo

Automate and centralize employee attendance with Hikvision biometric devices

This module allows you to synchronize employees and attendance records between Odoo and Hikvision biometric devices. It enables both single and bulk employee registration, real-time attendance download, and detailed attendance reporting.

Employee Registration in Odoo

To begin, register an employee in the Odoo system with their personal details, including: full name, department, gender, biometric ID, assigned device, and a formal photo if the device supports facial recognition.

Register Employees to Device

Employees can be uploaded to Hikvision devices using two methods:

  • Individual Upload: From the employee form, click “Upload user to device”.
  • Bulk Upload: Go to Attendance → Biometric Device → Hikvision Device and click “Upload all users to the device”.

Always ensure the device connection is working using the “Connection Test” button before uploading.

Download and Review Attendance

You can download attendance records from the device with a specified date and time range. This is done from the Hikvision device form using the “Download Attendances” button.

Generate Attendance Reports

After downloading attendance data, generate reports in Excel format from the Assistance menu. Select the records, click “Actions” → “Download attendance report”, and choose the time period and device.
